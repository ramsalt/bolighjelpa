
The Addresses Extras - Names module adds 4 fields to the Addresses module

This module adds the following:

  1. First name of the contact (i.e. Alexis)

  2. Last name of the contact (i.e. Wilke)

  3. Alias of the contact (i.e. Alex)

  4. Business name of the contact (i.e. Made to Order Software Corp.)

This is a good example on how the Addresses module can be extended.
