
This module is a set of sub-modules used to extend the Addresses
module with other fields and functionality that will not be
implemented in the Addresses module.

Sub-modules available:

  * Notes--offers a text field for general notes
  * Internet--offers common fields such as ICQ, Jabber, MSN, YIM, URL, email
  * Names--offers first, last, alias & business name
  * Country Selection--transforms the country selection to checkboxes

Please, check each module for more information.
